# ⚡️ `<RangesMarker />`

Sometimes you may want to mark text-content based on the
position in the text content instead of a string. For ex.
you would like to mark 5 letters from the 3rd position of text.

## [Stack Blitz Demo ↗](https://stackblitz.com/edit/react-mark-js?file=src/examples/RangesExample/index.js)

export const args = {
  mark: [
    {
      start: 3,
      length: 5,
    },
  ],
  children: <h1>0123456789</h1>,
};

## ‣ Basic Example

In this example, we mark 4 letters starting
from the 3rd letters.

<Canvas>
  <Story
    name="single-range-marker"
    args={{
      ...args,
      mark: [
        {
          start: 3,
          length: 5,
        },
      ],
    }}
  >
    {(args) => <RangesMarker {...args} />}
  </Story>
</Canvas>
