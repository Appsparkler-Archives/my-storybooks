# ⚡️ `useMarker` 
This recipe is for more custom configuration as you can 
access the `ref` itself.  We can pass the `options`
and `un-mark` options directly to this hook which will 
return a `markerRef` which can be passed to the wrapper 
element.

## [Stack Blitz Demo ↗️](https://stackblitz.com/edit/react-mark-js?file=src/examples/HookExample/index.js)

```js
import {useMarker} from 'react-mark.js'

const MyContent = () => {
    const markerRef = useMarker({
        options: {},
        unMarkOptions: {},
        type: 'mark' // or "markRanges" or "markRegExp",
        mark: "orl" // the text you want to mark
    })

    return (
        <div ref={markerRef}>
            Hello World
        </div>
    )
}
```
